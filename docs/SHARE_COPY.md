## 一句话介绍

用途水印是一个本地运行的 Codex Skill，可为证件、合同和其他文档图片批量添加“仅供某用途使用”的平铺文字水印。

## GitHub Description

```text
Offline Codex Skill for adding tiled purpose watermarks to document images.
```

## 功能摘要

- 本地处理，不主动上传图片
- 支持单图、多图和目录批量处理
- 支持默认、普通自定义和高级自定义三种流程
- Prompt 已给出的属性不重复询问
- 保持原图格式和尺寸，输出文件追加 `_watermark`
- 不覆盖原图

## 小红书正文

```text
做了一个“用途水印 / Purpose Watermark”Codex Skill。

它可以给身份证、证件照、营业执照、合同等图片添加“仅供某用途使用”的平铺文字水印。图片在本机处理，不需要上传到在线水印网站，处理后生成新文件，原图不会被覆盖。

直接提供图片和水印文案就能使用默认样式；说“自定义水印”可以设置颜色、透明度和字号；说“高级自定义”还可以调整角度与间距。Prompt 里已经写明的配置不会重复询问。

它不能保证绝对防盗用，但可以在提交材料前明确图片用途，减少裸传敏感图片的风险。
```
